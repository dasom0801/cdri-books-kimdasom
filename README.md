# CERTICOS BOOKS

## 프로젝트 개요

도서 검색, 상세 검색, 찜 목록 기능을 제공하는 웹 애플리케이션입니다.
도서명, 저자명, 출판사를 기준으로 상세 검색할 수 있으며, 원하는 책을 찜 목록에 추가해 별도 관리할 수 있습니다.

## 실행 방법 및 환경 설정

```bash
# 의존성 설치
npm install

# 로컬 개발 서버 실행
npm run dev

# 스토리북 실행
npm run storybook
```

## 라이브러리 선택 이유

- **react-router**
- **storybook**
- **sass**
